# ColorGame
## Instructions:
	1. User has to guess the color from the below options when given RGB value.
	2. There are 3 Modes to play. Easy, Medium and Hard. 

## Concepts used:
	* DOM Manipulation
	* Event Handling
	* CSS Transitions & Events
## Technologies Used:
	* HTML
	* CSS
	* Vanilla Javascript
